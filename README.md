# BitZealth — Complete Software Agency

> **"You have the idea. We handle everything else."**

BitZealth is a premium software development agency providing end-to-end solutions for ambitious founders. From initial discovery and UI/UX design to robust engineering, automated testing, and global deployment — BitZealth handles the entire technology lifecycle so you can focus on your vision.

## 🚀 Key Features

### 🌍 Global-Ready & Multilingual
Built with a sophisticated internationalization system supporting **English**, **Arabic (Full RTL)**, and **French**. The entire layout adapts seamlessly to different languages and reading directions.

### 🎨 Premium Design System
A sleek, modern aesthetic using a curated dark-mode palette, vibrant glassmorphism effects, and smooth micro-animations. BitZealth prioritizes **Visual Excellence** to deliver a "wow" factor at first glance.

### 🔍 AI-Ready SEO Strategy
Optimized for the next generation of search. Implements thorough **JSON-LD Schema**, semantic HTML, and metadata to ensure maximum visibility on Google and AI search engines.

### 🏗️ Data-Driven Architecture
- **Single Source of Truth**: All brand tokens, colors, and configurations are managed in a centralized `brand.js`.
- **Dynamic Content**: Portfolio, team profiles, and services are powered by structured JSON data, making the site incredibly easy to maintain and scale.
- **End-to-End Lifecycle**: Explicitly communicates the project roadmap from Discovery to Launch, building trust and transparency with clients.

---

## 💻 Tech Stack

- **Core**: React 18 + Vite 5
- **Routing**: React Router v6
- **i18n**: react-i18next (Multilingual & RTL Support)
- **Icons**: Lucide React
- **Styling**: CSS Modules with CSS Custom Properties (Native Variables)
- **Deployment**: Optimized for Cloudflare Pages (Global CDN)
- **Forms**: Formspree (Serverless Contact Handling)

## 🛠️ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
```bash
git clone https://github.com/bitzealth/BitZealthFrontend.git
cd BitZealthFrontend
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

---

## 🏛️ Architecture Highlights

- **`src/config/brand.js`**: Central repository for all brand identities. Changing a color or a logo here updates the entire site.
- **`src/data/`**: JSON-based content management for projects, team, and services.
- **`src/i18n/`**: Translation files for a truly global reach.
